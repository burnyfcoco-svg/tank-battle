# 🎮 坦克大战 (Tank Battle)

一个用纯 HTML/CSS/JavaScript 制作的经典坦克大战游戏，可直接在浏览器中运行。

## 🕹️ 游戏玩法

- **移动**: WASD 或方向键控制坦克移动
- **射击**: 空格键或 J 键发射炮弹
- **目标**: 消灭关卡中所有敌方坦克进入下一关
- **道具**: 拾取地图上的道具获得增益效果

## ⚡ 道具类型

| 道具 | 效果 |
|------|------|
| ⚡ 闪电 | 移动速度提升 60%，持续 8 秒 |
| 🔥 火焰 | 射击速度大幅提升，持续 8 秒 |
| 🛡️ 盾牌 | 获得无敌护盾，持续 5 秒 |
| ❤️ 红心 | 增加一条生命（最多 5 条） |

## 🎯 游戏特色

- 🧱 **可破坏砖墙** - 炮弹可以摧毁砖墙，开辟新路线
- 🛡️ **钢铁墙壁** - 无法被摧毁的障碍物
- 💥 **爆炸粒子特效** - 击中目标时的视觉反馈
- 📡 **小地图** - 右上角雷达显示敌我位置
- 🔊 **音效系统** - 射击、爆炸、道具拾取等音效
- 🏆 **关卡递增** - 每关敌人更多、更快
- 🗺️ **随机地图** - 每关地图布局不同

## 🚀 如何运行

1. 克隆仓库
```bash
git clone https://github.com/BURN/tank-battle.git
```

2. 用浏览器打开 `index.html` 即可开始游戏

或直接双击 `index.html` 文件在浏览器中打开。

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3
- 原生 JavaScript (ES6+)
- Web Audio API
